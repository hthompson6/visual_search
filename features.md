# Data Engineering

We have the following entities available

1. Images
2. Users
3. User-Image Interactions


## Image Metadata Table SQL

1. User UUID (owner)
2. Upload time
3. Description
4. Manual tags
5. Image S3 URL
6. Image UUID

## User Data Table SQL

1. Username
2. First Name
3. Last Name
4. Email
5. Phone Number
6. City
7. Country
8. Gender
9. DOB
10. UUID

## Image User Interaction Table NoSQL

1. User UUID
2. Interaction Type
3. Timestamp
4. List display rank
5. Image UUID
